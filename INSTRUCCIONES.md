# 📱 TIVASANI — Instalar como App

## Archivos que debes subir (todos)
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

---

## PASO 1 — Subir a GitHub Pages (gratis)

1. Ve a https://github.com → crea cuenta gratuita
2. Clic en "+" → "New repository"
3. Nombre: tivasani | Público | Sin README → "Create repository"
4. En la página del repo vacío, clic en "uploading an existing file"
5. Arrastra los 5 archivos → "Commit changes"
6. Ve a Settings → Pages → Branch: main → / (root) → Save
7. Espera ~1 min → tu URL será:
   https://TU-USUARIO.github.io/tivasani/

---

## PASO 2 — Instalar en Android

1. Abre Chrome en tu Android
2. Entra a la URL de GitHub Pages
3. Espera que cargue completamente (~3 seg)
4. Aparecerá un botón azul en la parte inferior:
   "📲 Instalar TIVASANI como app"
5. Tócalo → confirma → ¡listo!

Si el botón no aparece automáticamente:
• Menú Chrome (⋮) → "Instalar app" o "Agregar a pantalla de inicio"

---

## PASO 3 — Instalar en iPhone (Safari)

1. Abre Safari (no Chrome) en iPhone
2. Entra a la URL
3. Toca el ícono Compartir (□↑)
4. "Añadir a pantalla de inicio" → Añadir

---

## ¿Por qué necesita internet la primera vez?
La app descarga y guarda todo en el teléfono.
Después funciona SIN internet (excepto la radio en vivo).
